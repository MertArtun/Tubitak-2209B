# Basari Metrikleri

## Model Performansi
- Macro F1 >= 0.75 (FER2013 test seti uzerinde)
- Sinif bazli F1 raporu (negative, neutral, positive ayri ayri)

## Kimlik Esleme
- 5-10 kisilik kontrollu ortamda basarili esleme
- Cosine similarity threshold >= 0.5

## Sistem
- Demo sirasinda 5+ ogrenci ile kararli calisma
- Tek yuz analizi < 1 saniye (CPU)

## Demo Basarisi
- Dikkat skorlari anlamli (focused/moderate/distracted dagilimi mantikli)
- Excel export hatasiz uretilebiliyor
- Session akisi (baslat -> analiz -> durdur -> export) sorunsuz calisiyor
